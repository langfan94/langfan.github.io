查看npm解析安装依赖包详细情况

npm install –verbose (打印详细信息)
npm install –loglevel silly (答应更加具体的解析安装详情)
npm install –loglevel silly 2>&1 | tee log.txt (输出安装依赖信息到log.txt文件中 2>&1 输出成单个流; tee 读取标准输入到数据，并将内容输出成文件)
